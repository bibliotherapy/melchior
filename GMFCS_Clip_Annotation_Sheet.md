# GMFCS Clip Annotation Sheet

**One row per clip | Print on A4 landscape | Keep the code card below visible while annotating**

---

## Code Card (cut out and keep next to you)

```
DEVICE        ASSISTANCE              SURFACE
N  = None     1.0 = Independent       F = Floor
W  = Walker   0.8 = Standby           C = Chair
A  = AFO      0.6 = Minimal assist    - = N/A
S  = Acrylic  0.4 = Moderate assist
WA = Walker   0.2 = Maximal assist
     + AFO    0.0 = Fully dependent
```
