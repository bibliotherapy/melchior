## E. Independence Scale Quick Reference

*Use this table to convert the caregiver assistance checkbox to the numeric independence score.*

| Checkbox | Score | Description |
|:---------|:-----:|:------------|
| None (independent) | **1.0** | No assistance of any kind |
| Standby | **0.8** | Caregiver nearby, no physical contact |
| Minimal | **0.6** | Light touch, verbal cues, patient does >75% |
| Moderate | **0.4** | Physical support, patient does 25–50% |
| Maximal | **0.2** | Caregiver does >75% of the movement |
| *Fully dependent* | **0.0** | *Caregiver performs entire movement* |

## F. Mapping to Context Vector (18D)

*For pipeline reference. Researchers do not need to fill this section.*

```
Patient-Invariant Zone (11D):
  [0]  sex                → Section A
  [1]  age_normalized     → Section A (age_months / 72)
  [2]  w_status           → D-1 Status
  [3]  cr_status          → D-2 Status
  [4]  c_s_status         → D-3 Status
  [5]  s_c_status         → D-5 Status
  [6]  sr_status          → D-7 Status
  [7]  cc_s_status        → D-4 Status
  [8]  s_cc_status        → D-6 Status
  [9]  overall_assistance → Section C (score / 6)
  [10] device_count       → Section B (count of "Yes" / 3)

Clip-Specific Zone (7D) — filled per clip from the matching D-section:
  [11] clip_walker        → "Walker used?" in the matching D-section
  [12] clip_walker_type   → Section B walker type (if walker used)
  [13] clip_afo           → "AFO worn?" in the matching D-section
  [14] clip_afo_lat       → Section B AFO laterality (if AFO worn)
  [15] clip_acrylic_stand → "Acrylic stand used?" in the matching D-section
  [16] clip_surface       → 0=floor (c_s/s_c), 1=chair (cc_s/s_cc), 0.5=other
  [17] clip_assistance    → Independence score from the matching D-section
```
